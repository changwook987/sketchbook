# sketchbook
마인크래프트 스케치북 데이터팩입니다!
```
데이터 팩을 자신의 세이브 파일에 넣거나, 맵 생성시에 데이터 팩에 넣어야 합니다.
반복 명령블록에 /function setting:default 를 넣고 실행 시켜야 합니다.
```
```
발광석 가루를 주로 사용하지 않는 손(기본은 왼손)에 두면 노란색으로 그려집니다.
지우고 싶을 땐 종이를 주로 사용하지 않는 손에 두고 그림 가까이에 포인터를 가져다 대면 됩니다.
빨간 염료, 노란 염료, 검은 염료, 하얀 염료, 보라색 염료는 색깔을 바꿀 수 있습니다.
```
```
참나무 표지판을 주로 사용하지 않는 손에 두면 그룹으로 지정됩니다.
그룹으로 지정되면 바꾸고 싶은 염료를 들고 /function setting:group_color 를 입력하세요.
그룹을 풀고싶으면 /function setting:ungroup 을 입력하세요.
```
